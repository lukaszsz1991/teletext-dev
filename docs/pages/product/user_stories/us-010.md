# US-010: Zarządzanie stronami telegazety

#### Jako administrator, chcę mieć możliwość zarządzania stronami telegazety, aby móc dodawać i edytować treści. Zadanie: TELE1-62

**Kryteria akceptacji:**
- [x] Można dodać nową stronę z ogłoszeniem
- [x] Można edytować poprzednio utworzone strony
- [x] Nie można edytować stałych stron - integracyjnych
- [x] Przy tworzeniu strony można wybrać numer strony
- [x] Numer strony jest walidowany przed duplikatami
- [x] Strona musi posiadać: kategorię, numer, tytuł i opis
