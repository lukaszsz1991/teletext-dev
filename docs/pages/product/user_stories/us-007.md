# US-007: Zarządzanie użytkownikami

#### Jako administrator chcę mieć możliwość zarządzania innymi użytkownikami, aby utrzymać porządek i bezpieczeństwo aplikacji. Zadanie: TELE1-61

**Kryteria akceptacji:**

- [x] Można dodać, usunąć (soft-delete) i przywrócić użytkownika
- [ ] Można edytować istniejącego użytkownika
- [ ] Można zmienić hasło użytkownika
- [ ] Nie można usunąć siebie
- [ ] Nie zalogowani użytkownicy nie mogą wykonać powyższych operacji
- [ ] Operacje dostępne tylko dla roli ADMIN
