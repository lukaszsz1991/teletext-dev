# US-006: Cachowanie danych

#### Jako użytkownik końcowy, chcę wyświetlać dane z należytą szybkością, aby czerpać z aplikacji to, co w niej najlepsze. Zadanie: TELE1-87

**Kryteria akceptacji:**
- [ ] Najczęściej używane tabele powinny być ładowane z pamięci Cache
- [ ] Baza danych Redis powinna być dobrze skonfigurowana
- [ ] Dane powinny być zawsze aktualne
- [ ] Dane powinny być zgodne z rzeczywistością
