# US-005: Nawigacja użytkownika między stronami telegazety

#### Dodaj nawigację umożliwiającą użytkownikowi przechodzenie między kolejnymi numerami stron telegazety. Nawigacja w stylu klasycznej telegazety. Zadanie: TELE1-10

**Wymagania funkcjonalne:**

- Przyciski nawigacji:

  - "◄ Poprzednia" (pageNumber - 1)

  - "Następna ►" (pageNumber + 1)

- Input do wpisania numeru strony + przycisk "Idź"

- Walidacja czy numer strony istnieje

- Jeśli strona nie istnieje - komunikat błędu

- Nawigacja zawsze widoczna (na górze i dole strony)

**Wymagania techniczne:**

- Komponent: PageNavigation.jsx

- Wykorzystaj React Router do nawigacji (useNavigate, useParams)

- Sprawdzenie czy strona istnieje przed przejściem (mock API)
