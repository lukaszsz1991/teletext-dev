# US-011: Zarządzanie szablonami stron telegazety

#### Jako administrator, chcę mieć możliwość zarządzania szablonami stron tworzonych na podstawie integracji z zewnętrznymi API, aby dane były poprawne i aktualne. Zadanie: TELE1-76

**Kryteria akceptacji:**
- [ ] Można podejrzeć wszystkie szablony
- [ ] Można dodać nowy szablon
- [ ] Można edytować utworzone szablony
- [ ] Można zarchiwizować szablony, pod warunkiem, że nie mają aktywnej strony
- [ ] Można przywrócić szablon
- [ ] Szablon ma: kategorię, źródło, nazwę i pola konfiguracyjne
- [ ] Każde pole konfiguracyjne jest zależne od źródła
- [ ] Nazwa szablonu jest unikalna
