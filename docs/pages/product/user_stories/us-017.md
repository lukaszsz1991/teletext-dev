# US-017: Integracja z API Pogody

#### Jako użytkownik chcę widzieć tygodniową prognozę pogody dla wszystkich miast wojewódzkich na stronie, aby planować swoje aktywności. Zadanie: TELE1-15

**Kryteria akceptacji:**
- [ ] Użytkownik widzi pogodę dla wszystkich miast wojewódzkich w Polsce na następne 7 dni
- [ ] Pogoda jest zgodna z rzeczywistą prognozą
- [ ] Miasta można dodać przez plik csv, z którego dane są pobierane na bieżąco.
