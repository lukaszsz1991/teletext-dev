# US-013: Wysyłka maili

#### Jako administrator, chcę otrzymywać maile informujące o istotnych działaniach dotyczących mojego konta, aby być na bieżąco ze zmianami. Zadanie: TELE1-80

**Kryteria akceptacji:**
- Użytkownik dostaje wiadomość email nt.:
  - [ ] utworzenia jego konta
  - [ ] zmiany hasła na jego koncie
  - [ ] dezaktywacji jego konta
  - [ ] przywrócenia jego konta
