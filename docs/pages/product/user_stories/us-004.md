# US-004: Widok użytkownika - wyświetlanie strony telegazety

#### Stworzenie podstawowego widoku wyświetlania strony telegazety dla użytkownika końcowego. Strona powinna być wyświetlona w stylu retro telegazety z lat 80/90. Zadanie: TELE1-9

**Wymagania funkcjonalne:**

- Wyświetlanie pojedynczej strony telegazety po numerze

- Informacje: numer strony, tytuł, kategoria, treść

- Obsługa ASCII art w treści

- Jeśli strona ma przypisaną integrację - wyświetl placeholder "Integracja: [nazwa]"

- Stylizacja retro (CRT effect, monospace font, retro kolory)

- Obsługa błędów (strona nie istnieje)

**Wymagania techniczne:**

- Komponent: TeletextPageView.jsx

- API endpoint (mock): GET /api/pages/{pageNumber}

- Routing: /page/:pageNumber

- Stylizacja CSS w stylu telegazety (wykorzystaj to co już masz z kodu)
