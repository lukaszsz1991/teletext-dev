# US-008: Logowanie administratora

#### Jako administrator, chcę móc zalogować się do systemu, aby bezpiecznie móc zarządzać zawartością strony i innymi użytkownikami. Zadanie: TELE1-56

**Kryteria akceptacji:**
- [ ] Autoryzacja odbywa się poprzez JWT
- [ ] JWT musi być poprawnie generowane
- [ ] Użytkownicy nieaktywni nie mogą się zalogować
