# US-014: Integracja z API ofert pracy

#### Jako użytkownik chcę mieć wgląd w nowe oferty pracy, aby móc rozwijać swoją karierę. Zadanie: TELE1-20

**Kryteria akceptacji:**
- [ ] Użytkownik widzi oferty pracy z uwzględnioną lokalizacją i słowami kluczowymi
- [ ] Oferty pracy mają odniesienie do rzeczywistych ofert (link)
- [ ] Oferta pracy ma tytuł i krótki opis
