# US-001: Wdrożenie contentu strony telegazety

**Jako użytkownik chcę wyświetlić stronę telegazety z pełną nawigacją, aby łatwo przeglądać treści i przełączać się między stronami. Zadanie: TELE1-3**

**Kryteria akceptacji:**
- [ ] Góra strony zawiera panel nawigacji (data, godzina, numer strony, strzałki)
- [ ] Po prawej stronie widoczna lista kategorii z numeracją
- [ ] Widok poprawnie obsługuje brak treści strony
- [ ] API zwraca ujednolicony format strony (title, description, source, additional info)
