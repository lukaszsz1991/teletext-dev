# US-002: Dodanie endpointu stron z podziałem na kategorie

#### Jako użytkownik chcę pobrać listę kategorii wraz z ich numerami, aby móc nawigować po stronach telegazety i wyszukać intersujące mnie treści. Zadanie: TELE1-4

**Kryteria akceptacji:**
- [ ] API zwraca listę kategorii wraz z numerami
- [ ] Numer kategorii jest wielokrotnością 100 (max. 900)
- [ ] Walidacja: brak możliwości ustawienia numerów spoza zakresu
- [ ] Dokumentacja endpointów i kluczowych metod
