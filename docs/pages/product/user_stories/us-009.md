# US-009: Wyświetlanie statystyk stron

#### Jako administrator, chcę mieć wgląd w statystyki objerzeń poszczególnych stron, aby wiedzieć jakie treści przyciągają odbiorców. Zadanie: TELE1-63

**Kryteria akceptacji:**
- [ ] Wyświetlane dane są przejrzyste i proste w zrozumieniu
- [ ] Można podejrzeć szczegóły wejść z wyróżnieniem na dni
- [ ] Statystyki są poprawnie naliczane przy każdym odwiedzeniu
